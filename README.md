# ejemplo-node-ts

> Proyecto de ejemplo con node-ts funcionando con debugger para visual code

## Requerimientos

* nodejs

## Ejecucion

* Ejecutar `npm install`
* Ejecutar `npm run dev`

## Transpilar codigo

* Ejecutar `npm run build`
